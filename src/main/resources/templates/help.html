Welcome to the help page for the FileExplorer WebApp!


Purpose of This Software:


This software was made to create a user friendly file explorer system that utilizes already existing technology in browsers to display files. This was due to my dissatisfaction with current file explorers and the inability to add tags to files and search properly. Even though MacOS allows you to add tags, I dislike the rest of the UI to the point where it is unusable and I prefer using my terminal and spamming "cd." and "ls"


File Management:


Uploading Files:

Currently, you may only load a directory at a time, and any directory you select will have every file in it, and its subdirectories uploaded. To upload files, look at the top of the page at the bar that says "Directory to seed" and enter the absolute path of the directory you wish to seed. Please note that this must be an absolute path from your root directory. The directory traverser will recursively go through all of the files and directories in the initial directory provided and shortly update your view with all of your files. Once you see the file icons, you may click on any of them to begin viewing and interacting with them. Seeding a directory will redirect you to the home page, please keep this in mind while you are working with the software

Adding Tags to Files:

To add a tag to a file you have already uploaded, you must first click on the file and enter the page to view it. Below the displayed file will be a few options, below that will be a text box where you may enter your desired tags. You must separate each tag by a comma and none of your tags may contain the "|". Doing this will reload the page you are currently on and cause you to lose changes you've made, so please be careful character.

Searching Through Files:

Files are searched by either exact name or their tags. If you want to find a specific file, you must know its exact name, or tags unique to that file. Currently only one search term at a time is supported, but updates are coming. In order to search an item by tag, you must first add tags to this item.

Viewing Tags and Changing File Information:

In order to view the tags of a file or to change any of its information, click on the file from any page to go into its viewing page. From there, find the button that says "File Settings" and click on it to go to the specific video's page. From this page you can view the current tags and click on them to view other files with similar tags, plus change the title, location of the file on your device, and add tags.

Account Management:


The user has the option of making their own account if they have preferences they would like to save. Otherwise, this software has a "default" user upon first initialization which behaves like any other user, except is automatically logged in upon startup and requires no verification. If the user has ANY desire for privacy, they should make an account.

Creating an Account:

In order to create an account, hover your mouse over the menu button on the banner and select "Log In/Sign Up." There, you will be directed to a page where you can log into an already existing account or you can create a new one. Your account must have a username not used by any previous accounts.

Saving Your Account:

To save your account, make your way over to the menu button and click "Save Session." This will automatically save your account information for your use the next time you log in. NOTE: this will override any previous information, so files removed will be lost to the software if you save your session. NOTE: your information will only be saved if you manually save your session, otherwise you will not see any changes you made upon logging in again.
